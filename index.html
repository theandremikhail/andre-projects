AI Email Assistant

Built by Andre Mikhail Serra · Simulated demo of LLM-powered email replies

About the Project

This assistant was built to intelligently process incoming Gmail messages, classify their intent (e.g., invoice, job, meeting, support), and generate context-aware replies using a local LLM (LLaMA 3). The tool replicates how a smart email bot can help professionals automate inbox management securely and efficiently.

  • ✅ Gmail API + OAuth 2.0 integration
  • ✅ Threaded reply support with proper headers
  • ✅ Spam/no-reply email filtering
  • ✅ Local LLM (LLaMA 3) used for offline inference
  • ✅ Full HTML + MIME message formatting
  • ✅ Categorization: Invoice, Job, Meeting, Support, Spam
